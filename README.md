# CS 300 - Week 2 Lab

A Node.js project demonstrating GitHub Flow, branching, merge conflict resolution, and basic JavaScript functions.

---

## Description

This repository demonstrates a full Git workflow including feature branches, pull requests, and merge conflict resolution. The application in `src/app.js` includes simple user interaction functions such as greeting and farewell messages. It shows how professional developers collaborate using branches instead of committing directly to the main branch.

---

## Prerequisites

- Node.js v20 or higher
- Git
- A terminal / command line

---

## Git Workflow Used

This project followed a professional GitHub Flow workflow:

Created feature branches for each task
Used Conventional Commits for clear commit history
Pushed branches to GitHub
Opened Pull Requests for review and merging
Merged all changes into the main branch
Intentionally created and resolved a merge conflict between branches

This workflow ensures safe collaboration and prevents direct changes to the main branch.
